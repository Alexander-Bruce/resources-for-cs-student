## 仓库结构

当前仓库共整理 21 个顶层模块，约 416 个目录、1578 个文件。课程导览已按模块拆分到 `docs/`，每份页面都会先用代码块目录树展示结构，再说明各部分内容、代表文件和精确到文件的明细索引。

- `docs/`：按课程生成的导览页，包含带注释目录树、模块说明、入口链接和详细文件索引。
- `scripts/generate-docs.ps1`：重新扫描仓库并生成 README 与 docs 索引。
- `.gitignore`：统一忽略系统缓存、IDE 配置、编译产物和本地工具生成目录。
- `LICENSE`：MIT 许可证。

## 使用建议

- 查找某门课资料时，先在上方表格进入对应的课程导览。
- 查看实验、课程设计或源码时，优先从对应模块的 `实验`、`课程设计`、`源代码` 目录进入。
- 仓库已清理常见生成文件，源码类项目如需运行，请在本地按课程目录重新编译或安装依赖。

## 维护方式

新增、删除或移动资料后，可在 PowerShell 中运行以下命令刷新索引：

```powershell
./scripts/generate-docs.ps1
```
