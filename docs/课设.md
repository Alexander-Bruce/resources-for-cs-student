# 课设

[返回总览](../README.md) | [打开目录](../%E8%AF%BE%E8%AE%BE/)

## 概览

课设模块用于集中归档跨课程或综合类课程设计材料，适合快速定位独立项目和最终提交内容。

当前模块包含 0 个目录、7 个文件。建议先看目录结构，再从下方模块导览进入对应部分。

## 目录结构

```text
.
├── 详细设计说明书（案例）.docx                  # Word 文档
├── 软件工程任务书（ 基于Web应用的软件白盒测试用例生成系统） .doc # Word 文档
├── 软件工程任务书（基于Web应用的MCDC覆盖及评价系统）.doc  # Word 文档
├── 软件工程课程设计报告模板（个人）.docx             # 实验/课程报告
├── 软件工程课程设计报告模板（小组）.docx             # 实验/课程报告
├── 软件需求规格说明书（案例）.docx                # Word 文档
└── 需求规范.pdf                          # PDF 资料
```
