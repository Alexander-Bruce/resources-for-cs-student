### 原理分析：

#### 1. **线性回归的数学原理**

**线性回归** 是一种用于预测变量之间线性关系的统计方法。假设我们有一组数据，其中自变量（特征）为 X=(x1,x2,...,xn)X = (x_1, x_2, ..., x_n)X=(x1,x2,...,xn)，因变量为 y，线性回归模型的目标是找到一条最佳拟合线，使得预测值与实际值 之间的误差最小。

##### 线性回归的方程：

对于一元线性回归模型，其方程为：
$$
y=β0+β1⋅x1
$$
其中：

- β0 是截距，表示当自变量为零时因变量的预测值。
- β1 是斜率，表示自变量每变化一个单位时因变量的变化量。

对于多元线性回归（如本代码中的问题），其方程为：
$$
y=β0+β1⋅x1+β2⋅x2+⋯+βn⋅xn
$$
其中：

- β0 是截距。
- β1,β2,...,βn 是各特征  x1,x2,...,xnx_1, x_2, ..., x_nx1,x2,...,xn的回归系数。

##### 线性回归的目标：

通过训练数据找到最优的系数  β0,β1,...,βn，使得实际值 y和预测值之间的误差最小化。误差通常使用 **最小二乘法**来衡量。

#### 2. **最小二乘法 (OLS)**

最小二乘法是线性回归中用于估计回归系数的常用方法。其原理是通过最小化 **残差平方和**（RSS，Residual Sum of Squares）来找到最佳拟合线。残差是指每个数据点的实际值与预测值之间的差值，具体公式为：
$$
RSS=i=1∑n(yi−y^i)2
$$
其中：

- yi 是实际值
- yi 是预测值。

OLS 方法通过最小化上述残差平方和来估计回归系数  β1,β2,...,βn。

#### 3. **均方误差和均方根误差**

**均方误差**是衡量回归模型预测效果的重要指标之一，表示预测值与实际值之间差异的平均平方。公式为：
$$
MSE=n1i=1∑n(yi−y^i)2
$$
其中 n是样本数量，yi 是实际值，yi是预测值。

**均方根误差是均方误差的平方根，表示误差的标准偏差，常用来衡量模型预测的准确性。公式为：
$$
RMSE = √MSE
$$
RMSE 越小，模型的预测效果越好。

#### 4. **训练集和测试集的划分**

代码使用了 **train_test_split** 来将数据集分为训练集和测试集。这样做的目的是通过训练集来训练模型，然后用测试集来评估模型的泛化能力。

- **训练集**：用于训练模型，找出回归系数。

- ##### **测试集**：用于评估模型性能，确保模型不仅在训练数据上表现好，而且能在新的、未见过的数据上也有较好的预测性能。

### 代码分析：

#### 函数：读取数据并拆分为训练集和测试集
```
def getTrainSetAndTestSet(DataPath):
    data = pd.read_csv(DataPath)  # 使用pandas读取CSV文件，data是包含所有数据的DataFrame
    X = data[['AT','V','AP','RH']]  # 提取特征列：AT、V、AP和RH
    y = data[['PE']]  # 提取目标列：PE（能量输出）

    # 使用train_test_split将数据集随机划分为训练集和测试集，test_size=0.2表示80%用于训练，20%用于测试
	X_train, X_test, y_train, y_test = train_test_split(X, y, random_state=1)

    print("训练集和测试集的维度:")
    print(X_train.shape)  # 打印训练集特征的维度
    print(y_train.shape)  # 打印训练集目标的维度
	print(X_test.shape)   # 打印测试集特征的维度
	print(y_test.shape)   # 打印测试集目标的维度

    return X_train, X_test, y_train, y_test  # 返回训练集和测试集的数据
```



#### 函数：训练线性回归模型并输出回归参数
```
def TrainLinearRegression(X_train, y_train):
    linreg = LinearRegression()   # 初始化一个线性回归模型对象
    linreg.fit(X_train, y_train)  # 用训练集数据对模型进行训练

    # 输出线性回归模型的系数和截距
    print("线性回归各个系数(W)：", linreg.coef_)  # 输出模型的系数（权重）
    print("线性回归参数截距(b)：", linreg.intercept_)  # 输出模型的截距

	return linreg  # 返回训练后的模型
```

#### 函数：使用MSE（均方误差）和RMSE（均方根误差）评价模型的表现
```
def EvaluationModel(linreg, X_test, y_test):
    y_pred = linreg.predict(X_test)  # 用训练好的模型对测试集进行预测

    # 使用sklearn的metrics模块计算MSE
    mse = metrics.mean_squared_error(y_test, y_pred)  # 计算均方误差
    print("均方误差MSE:", mse)  # 打印MSE

    # 计算RMSE，RMSE是MSE的平方根
    rmse = np.sqrt(mse)  # 计算均方根误差
    print("均方根误差RMSE:", rmse)  # 打印RMSE

	return y_pred  # 返回预测结果
```



# 函数：可视化真实值与预测值的关系
```
def Visualization(y_test, y_pred):
    fig, ax = plt.subplots()  # 创建一个图形和坐标轴对象

    # 设置窗口标题
    manager = plt.get_current_fig_manager()  # 获取当前图形的管理器
    window = manager.canvas.manager.window  # 获取窗口
    window.winfo_toplevel().title("221002511贺庆林")  # 设置窗口的标题

    # 绘制散点图，横坐标是真实值，纵坐标是预测值
    ax.scatter(y_test, y_pred)  # 绘制真实值与预测值的散点图

    # 绘制黑色虚线，表示完美的预测（真实值等于预测值的情况）
    ax.plot([y_test.min(), y_test.max()], [y_test.min(), y_test.max()], 'k--', lw=5)  # k--表示黑色虚线，lw设置线宽

    # 设置坐标轴标签
    ax.set_xlabel("Measured")  # 横坐标标签：真实值
    ax.set_ylabel("Predicted")  # 纵坐标标签：预测值

    # 设置图表标题
    ax.set_title("线性回归模型", fontsize=14)  # 设置标题，字体大小为14

    # 显示图形
    plt.show()
```
